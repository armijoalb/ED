Ramas del repositorio:

- Master: rama principal, solo se suben los archivos finales ( los que vayamos a entregar ).
- Develop: rama para subir los archivos hechos para la práctica, probarlos y corregirlos.

Además, cada uno creará una rama para añadir alguna modificación deseada y luego se hará
un pull request a la rama develop. Al finalizar se hará un pull a la rama master cuando todo
funcione.

Para que la clase fecha compile correctamente es necesario compilar con C++11, para ello, añadid -std=c++0x al compilar
